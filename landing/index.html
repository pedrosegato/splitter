Como funciona

01

Abrir o Splitter

Inicie o app em cada máquina que vai participar. Nenhuma conta, nenhuma configuração de servidor.

02

Escolher a máquina na rede

O Splitter descobre automaticamente os peers na LAN. Selecione para onde rotear.

03

Rotear áudio

O áudio flui direto pela rede local, com latência mínima e sem intermediários na nuvem.

Requisitos

  • Rede local (LAN / Wi-Fi) entre as máquinas — sem internet necessária.
  • macOS: BlackHole 2ch para capturar áudio de sistema e expor microfone virtual.
  • Windows: VB-Cable para as mesmas funções de áudio de sistema e microfone virtual.
  • Linux: nenhum driver adicional — ALSA/PipeWire nativos.
  • macOS 12+, Windows 10+ ou Linux (glibc 2.17+).
