- **020 depends on 018 + 019** — the characterization tests lock current behavior so the control-plane extraction is provably behavior-preserving.
- **020 supersedes 003, 005, 015** — the unified control plane must fold in the lag fix (003), the notify/lookup dedup (005), and the disconnect eviction (015). If you do 020, either land those first and preserve them, or implement their fixes inside the unified module. 020's STOP conditions require reporting any CLI/Tauri divergence the observer can't express rather than silently picking one behavior.
- **014 + 015 coordinate** — both edit `crates/splitter-core/src/net/signaling/server.rs` (accept path). Land one, rebase the other.
- **005 + 015 + the `fix/disconnect-hard-kill` branch** — that branch already fixes the *manual* disconnect path (eviction + socket shutdown) in `src-tauri/src/commands/peers.rs`. 015 handles the *remote-drop* path in `acceptor.rs` + pre-auth eviction in `server.rs`; 005 leaves `teardown_session` out of scope. Merge the branch before starting 005/015 to avoid conflicts.

## Findings considered and rejected (non-findings)

- **`stream_id_counter` `wrapping_add`** (`session.rs:139`) — collision needs 256 concurrent streams; advertised `max_streams: 8`. Not reachable.
- **`let _ =` on broadcast `conn_est_tx.send` / `event_tx.send`** — a broadcast with no current subscribers is legitimately ignorable here.
- **`try_lock` in cpal callbacks** (`capture.rs`, `playback.rs`, `macos.rs`) — non-blocking, compliant with SAFETY.md #2. Correct as-is.
- **Path traversal via remote `device_id`** — device ids feed audio-device *enumeration*, never filesystem paths. All persisted paths derive from `dirs::config_dir()/data_dir()` + constant segments. No traversal.
- **Net-path panic-on-untrusted-input** — frame length bounded to 1 MiB; `Packet::decode` and `message.rs` guard before slicing; every wire `Uuid` parsed with a guarded fallback. No `unwrap`/index on socket bytes outside tests.
- **`app.security.csp = null`** (`tauri.conf.json`) — bundled local assets only; low risk. Cheap hardening, not a finding.
